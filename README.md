# java-web-gradle-spring-thyme-hello

## Description
A POC for a thyme springboot java gradle build.

## Tech stack
- springboot
  - web
- thymeleaf
- gradle
  - 6.8.2

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
